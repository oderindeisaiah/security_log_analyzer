# Security Log Analyzer

A defensive cybersecurity tool that analyzes authentication logs to detect
suspicious behavior such as brute-force login attempts.

## Features
- Parses authentication logs
- Detects repeated failed login attempts
- Flags suspicious IP addresses
- Generates security alerts

## Concepts Covered
- Log analysis
- Threat detection
- Blue team security logic
- Event-based monitoring

## Technology
- Python
- File handling
- Data analysis
